Detailed Description

template<class TIndex, class TValue>
struct mcgs::CSRAdaptor< TIndex, TValue >

Adaptor for matrices stored in the compressed sparse row format.

Template Parameters
TIndexInteger type of stored indices (int, long, unsigned or unsigned long).
TValueNumber type of stored entries (for now, only double is supported).

This class is a hollow adaptor that does not own any of the arrays associated with the matrix, and does not have mutable access to them. Its purpose is restricted to providing read access to the matrix.

See also
CSR format.
