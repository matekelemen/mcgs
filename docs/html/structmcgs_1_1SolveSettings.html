TValue mcgs::SolveSettings< TIndex, TValue >::residualAbsoluteTolerance

Absolute tolerance used for convergence checks.

Note
  • If both tolerances are non-positive, residuals are not computed and relaxation ends when the number of iterations reaches maxIterations.
  • Defaults to -1.

◆ verbosity

template<class TIndex , class TValue >
unsigned short mcgs::SolveSettings< TIndex, TValue >::verbosity

Verbosity level controlling the volume of output to std::cerr and std::cout.

Each level is a strict superset of lower levels.

  • 0: no output is written to std::cerr or std::cout.
  • 1: error messages are written to std::cerr.
  • 2: information outside iterations is written to std::cout.
  • 3: information from iterations is written to std::cout.
Note
Defaults to 1.
